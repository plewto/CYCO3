Configuration

Configuration is optional but recommended as it customizes CYCO to the local environment. It is possible to create multiple configurations for specific needs.

There is no prescribed structure to the configuration process other then it loads the file ~/.config/cyco/cyco-config.lisp. Typical uses are to assign MIDI channels and describe available external gear. See Instruments.

The folder CYCO/config/ contains two example configuration schemes. The default is a basic scheme based on generic General MIDI   instruments.

The second example called sj  is my personal configuration. See the file CYCO/config/README for more details.
