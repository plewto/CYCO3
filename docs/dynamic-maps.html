Dynamic Maps

A dynamic-map   is a function which defines an instrument's dynamic response. These maps may be used to reduce or boost velocity levels of overly loud/quite instrument.

(lambda d) --> float
(BASIC-DYNAMIC-MAP &key scale min max)

Returns a simple dynamic-map function.

:scale  - float scaling factor, default 1.0
:min :max - limits the range of results, defaults 0 and 1 respectively.
            The range test is applied post scaling.
(METRONOME-DYNAMIC-MAP &key phrase bar beat)

Creates dynamic-map function for use with metronomes.

+DEFAULT-DYNAMIC-MAP+

This constant holds the dynamic map used by default.

Articulation Maps

An Articulation-Map  is a function which defines an instrument's response to note duration.

(lambda m &key time-scale) --> float

m - a metric value
:time-scale - scales result based on current tempo.
              Returns float, the note on time.
(BASIC-ARTICULATION-MAP &key scale min max)

Creates a simple articulation map.

:scale - on time scaling factor, default 1.0
:min - minimum on time, default 0.
:max = maximum on time, default 1e6 (an arbitrarily large value)
(CONSTANT-ARTICULATION-MAP mxp)

Creates articulation-map with constant response, the mxp argument is a metric-expression.
