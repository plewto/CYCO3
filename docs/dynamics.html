Dynamics

Numeric Dynamic Values

Numeric dynamic values are normalized floats in the range 0.0..+1.0 inclusive. Non-positive values are treated as rest.

Symbolic Dynamic Values

Symbolic values use the common music symbols of P, MF, FF, etc.. These are extended with '-' and '+' signs to for extended resolution. A partial list of values follows

R     -1.0
PPPP-  ~ 0.016
pppp   ~ 0.047
PPPP+  ~ 0.079
PPP-   ~ 0.110
...      ...
MP-    ~ 0.417
MP     ~ 0.457
MP+    ~ 0.489
MF-    ~ 0.520
MF     ~ 0.560
MF+    ~ 0.590
...      ...
FFFF-  ~ 0.929
FFFF   ~ 0.969
FFFF+  ~ 1.000

Dynamic Functions

Many dynamic related functions may take a single value or a list of values.

(DYNAMIC obj)

Returns numeric dynamic value or list of values.

(DYNAMIC-P obj)

Predicate, true is argument is a valid dynamic value.

(DYNAMIC-NAME obj)

Returns the symbolic name for a dynamic value. There may be slight rounding errors during the conversion. This function is horribly inefficient.

(DEFDYNAMIC symbol value)

Define a user dynamic value.

(DYNAMIC->VELOCITY)
(VELOCITY->DYNAMIC)

These two functions Convert between normalized dynamic values and MIDI velocity.
