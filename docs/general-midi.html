General MIDI Plugin

The general-midi plugin  is a bare-bones generic MIDI profile using the General MIDI  standard.

To load:

CYCO: (plugin 'general-midi)

The general-midi plugin provides the following functions and values.

(MAKE-GENERAL-MIDI-INSTRUMENT name &key ...)
(GENERAL-MIDI-INSTRUMENT name &key ...)

Create new general-midi instrument. These two functions are identical except GENERAL-MIDI-INSTRUMENT binds the new object to the symbol name.  The following two lines are equivalent:

(param foo (make-general-midi-instrument 'foo  ...)) ;; name must be quoted
(general-midi-instrument foo ...)	             ;; name must -not- be quoted

name      - Instrument name
:program   - MIDI program number. If name matches an entry in +GENERAL-MIDI-PROGRAMS+
             the corresponding program number is used. 
:parent    - default +ROOT-INSTRUMENT+
:transient - boolean, default t
:channel   - MIDI channel number, defaults to parent channel
:keynumber-map    - function.
:dynamic-map      - function.
:articulation-map - function.
:remarks  - Optional remarks text
+GENERAL-MIDI-PROGRAMS+

Constant association list, maps program names to MIDI program numbers.

(?GENERAL-MIDI-PROGRAMS)

Displays list of general MIDI programs.

(GENERAL-MIDI-PROGRAM p)

Returns MIDI program number for general-midi program p.

If p is an integer and 0 ≤ p ≤ 127, return p.
If p is a key in +GENERAL-MIDI-PROGRAMS+, return associated program number.
Otherwise display warning and return 0.
+GENERAL-MIDI-PERCUSSION-KEYMAP+

Constant association list, maps percussion instrument names to MIDI key-number.

GM-PERCUSSION

Parent for all general-midi percussion instruments.

+GM-COWBELL-KEYMAP+
GM-COWBELL

General MIDI cowbell instrument and keymap function

+GM-WOODBLOCK-KEYMAP+
GM-WOODBLOCK

General MIDI woodblock instrument and keymap function

+GM-CYMBAL-KEYMAP+
GM-CYMBAL

General MIDI cymbal instrument and keymap function

+GM-DRUM-KEYMAP+

Instrument tree structure:

+ROOT-INSTRUMENT+	 
  |
  +-- gm-percussion
  |    |
  |    +-- gm-cowbell
  |    +-- gm-woodblock       
  |    +-- gm-cymbal
  |    +-- gm-drum
  |    +-- gm-hihat
  |    +-- gm-shaker
  |    +-- gm-kick
  |    +-- gm-cuica
  |    +-- gm-timbale
  |    +-- gm-tom
  |

Example Usage

The following creates two


(general-midi-instrument honkytonk)

    ;; Creates new instrument and bind it to the symbol HONKYTONK.
    ;; As honkytonk is a key in +GENERAL-MIDI-PROGRAMS+, the program 
    ;; number is automatically set, in this case to 3.

(general-midi-instrument pizz :program 'strings-pizz)

    ;; As above but name is different from program-number.
    ;; The value for :program may be an integer MIDI program-number
    ;; or a general-midi instrument name.
