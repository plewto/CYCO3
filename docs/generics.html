(->CYCLE object)

Coerce object to cycle pattern.

    Methods
    (->CYCLE t)       Return new cycle with object as it's element.
    (->CYCLE vector)  Return new cycle with vector contents as elements.
    (->CYCLE list)    Return new cycle with list contents as elements.
    (->CYCLE cycle)   Return object.
    (->CYCLE pattern) Return new cycle with elements from pattern.
