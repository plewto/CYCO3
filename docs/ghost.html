Ghost Parts

A GHOST  is a type of PART  which tracks another part.  The ghost events may be transposed, inverted, shifted in time, amplitude scaled or mapped to another MIDI channel.

Only note events are tracked and only a single MIDI channel of the source part is tracked.

(MAKE-GHOST name source track &key ...)
(GHOST name source track &key ...)

Create a new Ghost Part. MAKE-GHOST and GHOST are nearly identical, the only difference is that GHOST binds the new part to a symbol with the same name. The following two lines are equivalent.

(param foo (make-ghost 'foo source track ...)   ;; name must be a quoted symbol 
(ghost foo source track ...)	                ;; name must -not- be quoted.

Arguments

name       - Symbol
source     - Part, the part being tracked
track      - MIDI channel or instrument occurring in source.
             Only a single MIDI channel is tracked.
:delay     - Delay time of ghosted events.   The delay is specified in terms
             of the parent section's time-signature and cue-function. 
             It is NOT in terms of the tracked part.
:transpose - Transpose amount in half-steps.
:pivot     - Inversion pivot key.
:ampscale  - Amplitude scale, default 1.0
:keytab    - An array of keynumber translations, default +DEFAULT-KEYTABLE+
:outchan   - The ghosted events MIDI channel.  If nil use same value as track
:remarks   - Optional remarks text.
