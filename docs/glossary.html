CYCO Glossary

A MIDI channel designator is any object which may specify a MIDI channel. Typically these are either an integer, a symbolic channel name or an instrument.

A Normalized Value is a floating point number with an absolute value less then or equal to 1.0. 

    Unsigned normalized value   0.0 <= value <= 1.0
     
    Signed normalized value    -1.0 <= value <= +1.0
