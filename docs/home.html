CYCO is a Lisp based music composition language.  Scores are prepared by creating Projects which generate MIDI files to be played by an external MIDI file player.

CYCO began as a set of extensions for Common Music 2 called 'CMU' (for Common Music Utility). CMU added higher level structures which better suited my composition style. Eventually Common Music moved to version 3 and it became increasingly difficult to port CMU with each operating system update. In reality CMU used a small subset of Common Music features and it was decided to implement these from scratch, severing the dependence of Common Music.

Features

  • Project centric design
  • Flexible functional time-signature support
  • Algorithmic pattern generators
  • Symbolic names for common musical entities such as keynumbers and dynamics
  • Metric Expressions allow complex specification for note-durations.
  • Chord strumming patterns
  • Recombnitive patterns
  • Flexible instrument definitions
  • Plugins
  • CYCO's state may be saved to executable snapshots.
