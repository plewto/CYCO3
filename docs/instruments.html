CYCO Instruments

An instrument is a CYCO-NODE for defining properties of avaliable MIDI equipment.
CYCO does not define an orchestra object parse, instead the term 'orchestra' refers to the tree rooted at *ROOT-INSTRUMENT*. All instruments are linked to this tree.

Instruments are roughly divided into two groups:

  1. Non-transient, permanent instruments representing available external MIDI resources, either hardware or virtual.
  2. Transient, project level instruments defined as child nodes of the non-transient instruments.

See Nodes for an explanation of this division.

Instrument Properties

  • program-map
  • program-number
  • program-bank
  • keynumber-map
  • dynamic-map
  • articulation-map
  • MIDI channel
(INSTRUMENT-P object)

Predicate, true if argument is an instrument.

(PROGRAM-MAP! instrument function)
(PROGRAM-MAP instrument)

Sets/Retrieves instrument program map.

(PROGRAM-NUMBER! instrument program)
(PROGRAM-NUMBER instrument)
(PROGRAM-BANK! instrument bank)
(PROGRAM-BANK instrument)

These functions set or return an instrument default program number and bank.

(PROGRAM-CHANGE-EVENTS instrument time &key bank program)

Generates list of MIDI events to execute a program change.

    Unless overridden by the bank and/or program arguments, the
    instruments default bank/program properties are used.
(KEYNUMBER-MAP! instrument function)
(KEYNUMBER-MAP instrument)

Sets/Retrieves instrument keynumber map function.

(ARTICULATION-MAP! instrument function)
(ARTICULATION-MAP instrument)

Sets/Retrieves instrument articulation map function.

(DYNAMIC-MAP! instrument function)
(DYNAMIC-MAP instrument)

Sets/Retrieves instrument dynamic map function.

(CHANNEL! instrument channel)

Sets instrument MIDI channel.

    By default an instrument inherits the
    MIDI channel from its parent.  The channel may either be an absolute
    integer between 1 and 16 or a symbolic channel name.
(CHANNEL instrument &optional resolve)

Returns the instrument MIDI channel.

(CHANNEL-INDEX instrument)

Returns the instrument channel-index, an integer between 0 and 15.

*ROOT-INSTRUMENT*

Forms the root of the orchestra tree.

NULL-INSTRUMENT

place-holder instrument, does not generate any events.

(INSTRUMENT name &key ...)
(MAKE-INSTRUMENT name &key ...)

Creates new INSTRUMENT instance.

    INSTRUMENT AND MAKE-INSTRUMENT are identical except the former binds
    the new instrument to the the symbol name, while the later does not.
    The name argument should be unquoted for INSTRUMENT and quoted for
    MAKE-INSTRUMENT.
    
    	name              - Symbol 
    	:parent           - The parent instrument, default *ROOT-INSTRUMENT*
    	:transient        - Boolean, set to nil if the instrument is being defined by a project.
    	                    default nil.
    	:channel          - MIDI channel, defaults to parent channel.
    	:program          - Default program number, passed as :program argument to the program-map.
    	:bank             - Default program bank, passed as :bank argument to the program-map.
    	:keynumber-map    - Function, see keynumber-map
    	:dynamic-map      - Function, see dynamic-map
    	:articulation-map - Function, see articulation-map
    	:remarks          - Optional remarks text.
