Key Numbers

Numeric key Numbers

CYCO use the standard MIDI key-numbers 0..127 but treats all negative values as rest. The canonical rest value is -1.

Symbolic Key-numbers

Symbolic key-numbers have the form:    root-key + optional accidental + octave.

Where root key is one of: C D E F G A or B .
Optional accidental is either F (flat) or S (sharp).
Octave number is between 0 and 10 inclusive.

A few examples:
AS4 = BF4 = key-number 58
B4 = key-number 59
C5 = key-number 60, 'Middle C'
CS5 = DF5 = key-number 61

Enharmonic values are defined for b-sharp, c-flat, e-sharp and f-flat:
BS4 = CF5 and ES5 = FF5

The symbol R is a rest with value -1.

Keynumber functions

(KEYNUMBER obj)

Returns keynumber value of argument.

(keynumber 34)   --> 34
(keynumber -123) --> -1  a rest
(keynumber 'r)   --> -1
(keynumber 'c4)  --> 48

Most key-number related functions may be called with a nested list argument.
(keynumber '(20 30 40 (c4 c5 c6)) --> (20 30 40 (48 60 72)))
(KEYNUMBER-P obj)

Predicate, true if argument is a key-number.

(REST-P obj)

Predicate, true if argument is a key-number with negative value.

(PITCH-CLASS obj)

Returns integer pitch-class of key-number or list of key-numbers. The pitch class is the key-number mod 12.  All negative values have pitch class of -1.

(OCTAVE obj)

Returns octave number of key-number or list of key-numbers.  All negative key-numbers have octave value -1.

(KEYNAME obj)

Returns symbolic name of key-number or list of key-numbers.  All negative values have name 'R.
For key-numbers with more then one name, i.e. CS5 and DF5, the sharp version is returned.

(TRANSPOSE key n)

Transposes key-number or list of key-numbers by n steps.
Rest are always transposed to -1.
Out of bound results are automatically shifted to pitch-equivalent in-bounds value. The n argument may be nil

(INVERT key pivot)

Apply key inversion on key-number or list of key-numbers. The pivot point maybe an integer, symbolic-keynumber or nil.

(DEFKEYNUMBER symbol value)

Define user key-number.
value should be an integer between -1 and 127 inclusive.
The keyname function does not recognize user-defined key-numbers.
