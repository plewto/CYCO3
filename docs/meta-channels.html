MIDI Channels

Meta Channels

(META-CHANNEL! name channel &optional remarks)

Makes symbolic assignment to MIDI channel. A Meta channel may be defined in terms of another channel so long as they ultimately point to a 'real' channel number.

name - symbol
channel - integer, 1 ≤ channel ≤ 16
          symbol, an indirect assignment
remarks - optional remarks text.

Example:

(meta-channel! 'oberheim 1)
(meta-channel! 'drum-machine 10)
(meta-channel! 'snare 'drum-machine)

The snare channel is defined indirectly via the drum-machine.

(META-CHANNEL chan &optional (resolve nil))

Returns value of meta channel. By default returns one-level of indirection. If resolve is true, returns the actual MIDI channel.

(meta-channel 'snare) --> 'drum-machine
(meta-channel 'snare t) --> 10
(CHANNEL-NAME chan)

Returns the primary  name for MIDI channel.

(channel-name 10) --> 'drum-machine	 
(channel-name 6) --> 6  unassigned channels return their integer value.
(META-CHANNEL-ASSIGNMENT-P obj)

Predicate, true if obj is a channel assignment.

(?META-CHANNELS)

Displays list of channel assignments
