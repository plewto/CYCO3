Metrics

Metrics are relative values used for note durations. The basic metric values are

W     4    - Whole note
H     2    - Half
Q     1    - Quarter
E     1/2  - Eighth
S     1/4  - Sixteenth
T     1/8  - Thirty-second
X     1/16 - Sixty-fourth
R     -1   - A rest, has no intrinsic duration.

Dotted notes are formed by appending any number of dots  '.'  to the basic symbol.  Each dot scales the value by 3/2.

Q.     9/4  - dotted quarter note
Q..    27/8 - double dotted quarter note
Q...   81/16

Triplets are formed by appending 't'.   Each t scales the value by 2/3.

QT    2/3 - quarter note triplet
QTT   4/9 
QTTT  8/27

Combining dot and t modifications is legal but pointless.

QT.  1 - The t and . cancel

It is legal but pointless to use either dot or t with the rest symbol R.   A rest has no intrinsic duration and modifications have no effect.

Metric Functions

(METRIC-p obj)

Predicate, true if obj is valid metric value.

(METRIC obj)

Returns metric value of argument, may take either a single argument or list of arguments.

(metric 3.4)  --> 3.4
(metric -123) --> -1  All negative values treated as rest=-1.
(metric 'Q)   --> 1.0 quarter note
(metric 'E.)  --> 0.75 dotted eighth note
(metric 'ST)  --> ~0.1667 sixteenth note triplet.

Metric Expressions

A Metric-Expression  allows for complex durations by combining the basic metric values above.  The basic form is:

[s*]a[-+b][-+c]...

Where s is optional numeric scale factor and 
a,b,c are basic metric values as above.

Some Examples

W+Q    - add quarter and whole notes.
W-S.   - subtract dotted sixteenth from whole note.
W+Q+S  - Add whole, quarter and sixteenth notes.

The scaling factor is applied after all additions and subtractions.

3*W+S  - Add sixteenth note to whole note and -then- scale by 3.
(METRIC-EXPRESSION-P obj)

Predicate, true if argument is valid metric expression.

(METRIC-EXPRESSION obj)

Evaluate object, or list of objects, as metric expression.
