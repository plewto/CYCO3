Miscellaneous Topics

The function (? item) prints information about item.  The exact result depends on the item type.  useful arguments:

  • (? *PROJECT*)
  • (? section)
  • (? part)
  • (? function) - display function's documentation.

Other similar functions include:

  • (?D symbol) - shortcut for DESCRIBE.
    ?D displays documentation for most CYCO items.
  • (?A symbol &optional (package :CYCO)) - shortcut for APROPOS
  • (?CHANNELS) - displays meta channel assignments.
  • (?CHORDS) - displays list of defined chords in *CHORD-TABLE*

(??) list help functions
(?o) display orchestra tree
(?p) display projhhect structure
