Nodes

A CYCO-NODE is a low-level object used to build structures behind the scenes. A node has at most one parent, and any number of child nodes.

Each Node type define a pre-established set of properties and a node may either directly set a property value or inherit the value from it's parent.

CYCO-NODES are used to construct two major types of tree.

  1. Orchestra - rooted at +ROOT-INSTRUMENT+
  2. Current project - rooted at *PROJECT*

See Instruments  and Projects

While Nodes where not explicitly designed for the use by the user, they could conceivably have applications for algorithmic composition.

Node Related Functions

(NAME! node new-name)
(NAME node)

Sets and returns node's name

(REMARKS! node text)
(REMARKS node)

Sets and returns optional remarks text.

(ROOT-P node)

Predicate, true if node is a root node.

(FIND-CHILD parent child)
(FIND-CHILD parent child-name)

Returns child node from parent. Returns nil if child is not a child of parent.

(PATH-TO-ROOT node)

Returns a list of nodes from node back to the root node.

(DISCONNECT node)

Sever the connection between node and it's parent.

(CONNECT parent child)

Make child a child node of parent. The connection between child and it's previous parent is severed.

(PRUNE node &optional (force nil))

Deconstruct the tree starting with node. Node objects have a Boolean TRANSIENT   value.   Prune will only remove nodes whose transient value is true. If the optional force argument is true, then all nodes are removed regardless of their transient status.

(HAS-PROPERTY-P node key)

Predicate, true if node has the given property.

(PUT node key value)

Sets node key property to value. It is an error if node does not define key as a property. The new value will shadow the parent's value for key.

(PROPERTY node key)

Returns node's value for property key. If node does not define a value for key, retrieve the value from the parent node. It is an error if key is not an established property of node.

(PROPERTIES node)

Returns list of all property keys established for node.

(LOCAL-PROPERTIES node)

Returns list of property keys explicitly established for node, parent properties are ignored.

(PRINT-TREE node &optional (depth 0))

Prints a representation of the tree structure starting at node. The optional depth argument is used internally.
