OSC

CYCO uses zzkt/osc and  usocket  for rudimentary OSC  support.

The implementation is very basic and consist of a single CYCO function.  Currently only transmission is supported. It is unlikely OSC reception will be implemented, but it could happen.

(OSC-SEND arguments &key host port address)

Transmit a single OSC message.

arguments - list of contents. As a convenience a single item may be specified.  
             Use nil if there is no content.	 
:host - OSC server host address, defaults to *DEFAULT-OSC-SEND-HOST* 
        Initially the value is local-host.
:port - OSC server port number, defaults to *DEFAULT-OSC-SEND-PORT*
        Initially the default port is 65000, but this may change.
:address - The message address

The motivation for including OSC is to allow control over an external MIDI file player from within CYCO. A suitable MIDI file player, called PigIron, was developed concurrently with CYCO.
