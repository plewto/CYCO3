OSC Plugin

The OSC plugin  provides basic Open Sound Control  support.

Before using OSC you must first install quicklisp as per instructions in the link. Specifically quicklisp is expected to be at the default location ~/quicklisp/

The following two packages are imported by quicklisp:

  • zzkt/osc
  • usocket

The implementation is very basic and consist of a single CYCO function.  Currently only transmission is supported.
It is possible, but unlikely, that OSC reception will be implemented at some future point.
Implementing reception will probably involve creating a separate thread.

(OSC-SEND arguments &key host port address)

Transmit a single OSC message.

arguments - list of contents. As a convenience a single item may be specified.  
             Use nil if there is no content.	 
:host - OSC server host address, defaults to *DEFAULT-OSC-SEND-HOST* 
        Initially the value is local-host.
:port - OSC server port number, defaults to *DEFAULT-OSC-SEND-PORT*
        Initially the default port is 65000, but this may change.
:address - The message address
