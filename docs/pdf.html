PDF Plugin

Random Distributions

The PDF plugin provides several random distribution functions.  Much of the core functionality was ported from Nyquist.

For each distribution 'foo', two functions are provided:

  • foo-dist - Returns random numbers with the distribution foo.  These functions take one or more keyword arguments.
  • foo-pdf - Create a new function with distribution foo.  foo-pdf takes the same arguments as foo-dist and also arguments which place limits on the results.  The resulting function takes no arguments.

All foo-dist and foo-pdf functions take a :RNF keyword argument to set the random number generator.  The default :RNF is #'random

All foo-pdf functions take the keyword :MINMAX to restrict the possible numeric range.  If specified the value for :MINMAX must be a list of form (min max).  The default is to not restrict the results.

(linear-dist (&key (scale 1.0)(rnf #'random))
(linear-pdf (&key (scale 1.0)(rnf #'random) minmax)

(exponential-dist (&key (delta 2)(rnf #'random))
(exponential-pdf (&key (delta 2)(rnf #'random) minmax)

(gamma-dist (&key (mu 5)(rnf #'random))
(gamma-pdf (&key (mu 5)(rnf #'random) minmax)

(bilateral-dist (&key (mu 0)(tau 1)(rnf #'random))
(bilateral-pdf (&key (mu 0)(tau 1)(rnf #'random) minmax)

(cauchy-dist (&key (tau 1.0)(rnf #'random))
(cauchy-pdf (&key (tau 1.0)(rnf #'random) minmax)

(hyperbolic-cosine-dist (&key (rnf #'random))
(hyperbolic-cosine-pdf (&key (rnf #'random) minmax)

(gaussian-dist (&key (mu 0)(sigma 1)(rnf #'random))
(gaussian-pdf (&key (mu 0)(sigma 1)(rnf #'random) minmax)

(beta-dist (&key (a 1)(b 1)(rnf #'random))
(beta-pdf (&key (a 1)(b 1)(rnf #'random) minmax)

(bernoulli-dist (&key (p 0.5)(a 1)(b 0)(rnf #'random))
(bernoulli-pdf (&key (p 0.5)(a 1)(b 0)(rnf #'random) minmax)
