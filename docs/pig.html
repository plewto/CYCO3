Pig

Pigiron is a MIDI routing utility capable of playing MIDI files. The :PIG package defines several functions for remote access to Pigiron via OSC. Only those functions related to the MIDI file player are imported into the main :CYCO package. This page only details player related functions in the main :CYCO package and a few non-imported functions in the :PIG package. See the source for more general information.

With exception to the SET-PIG-SERVER and SET-PIG-PLAYER-ID, all other functions require that Pigiron is running and with an active MidiPlayer operator.

(SET-PIG-SERVER :host :port :id :response-filename :read-delay)

Establishes details about the Pigiron server.

    
    :host - IP address or host name, defaults to local machine #(127 0 0 1)
    :port - Defaults to 7000
    :id   - The OSC address prefix for the server, defaults to 'pig'.
    :response-filename - Location where Pigiron writes its response file. 
                         The contents of this file is checked after each transmitted
                         OSC message to determine success or failure.  The 
                         file also contains any requested information. 
                         Defaults to '/tmp/pigiron-response'
    :read-delay - Delay interval between transmitting an OSC message and checking
                  the response file.  This is to ensure Pigiron has had enough time
                  to write the response.  Defaults to 0.2 second.
(SET-PIG-PLAYER-ID id)

Sets the OSC id for the Pigiron MIDI player. If not explicitly changed the default id is 'player'.

(ADD-MEDIA-DIRECTORY directory-name

Adds all MIDI files in the directory to the Players Media list. Files must be added to the Media list before they may be played.

(CLEAR-MEDIA-LIST)
(DUMP-MEDIA-LIST)

Print contents of the Player's media list.

(SELECT-MEDIA name)

Selects named media for playback.

(PLAY)

Start playback of the selected media from the beginning.

(STOP)

(RESUME)

Continue playback from current position.

(IS-PLAYING)

Predicate, returns true if MIDI player is currently running.

Pig functions not imported into :CYCO

(PIG:PING)

Diagnostic used to test if the OSC connection between CYCO and Pigiron is working.
