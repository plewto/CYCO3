Preroll Sections

Preroll is a specialized section for initializing MIDI programs and controller values. It also contains a count-in metronome.

(MAKE-PREROLL &key ...)
(PREROLL &key ...)
    MAKE-PREROLL and PREROLL are identical except the later binds the new
    instance to the symbol name while the former does not.   The name argument
    should be quoted for MAKE-PREROLL and unquoted for PREROLL.
    
        Keyword arguments
    
        :name        - Sections name, defaults to 'PREROLL
        :instruments - List of instruments.  Program and bank change events are
                       generated for each instrument.
        :metronome   - Boolean, if true include a metronome, default t.
        :remarks     - Optional remarks text.
        :project     - Defaults to *PROJECT*
        :tempo       - Defaults to project's tempo. 
        :unit        - Defaults to project's unit.
        :bars        - Defaults to project's bars.
        :beats       - Defaults to project's beats.
        :subbeats    - Defaults to project's subbeats.
