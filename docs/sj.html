SJ MIDI Profile

The SJ MIDI profile is included for illustration purposes only.  It is the profile I use for my personal use and unlikely to be of much use to anyone but myself.  The sj-main file first defines a bunch of MIDI channels and then loads specific instruments.  There are folders for each synth manufacture and specific instruments may have their own sub-folder.  There is no specific reason to organize the folders this way, but it seems to make sense.
