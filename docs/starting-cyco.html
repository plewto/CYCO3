Starting CYCO

These symbols are used in the following examples.


 $     - Command line prompt.
 *     - Lisp REPL prompt.
 CYCO: - Lisp REPL prompt while using CYCO.

The first time CYCO is started, or anytime you wish to change the configuration, do the following.

Change directory to wherever the CYCO distribution folder is located.

$ cd ~/CYCO

Start your preferred Lisp,  here SBCl is assumed.

$ sbcl

At the lisp prompt load the cyco source files

* (load "src/cyco")
After the source files load, switch to the CYCO package.

* (CYCO)

While using CYCO the Lisp prompt changes to :CYCO.  The exact prompt text
changes to reflect the current project name.

Optional Configuration

Configuration is optional but highly suggested. See examples directory for details.
Use the (load-config) function to load the configuration files.

CYCO: (load-config)

Snapshots

Once CYCO is configured you may wish to create an executable snapshot.

CYCO: (save-snapshot "name-of-snapshot")

After creating the snapshot CYCO with terminate. This is normal.
