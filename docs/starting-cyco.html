Starting CYCO

These symbols are used in the following examples.


 $     - Command line prompt.
 *     - Lisp REPL prompt.
 CYCO: - Lisp REPL prompt while using CYCO.

To start CYCO:

Change directory to wherever the CYCO distribution folder is located.

$ cd ~/CYCO

Start your preferred Lisp (SBCl is assumed).

$ sbcl

At the lisp prompt load the cyco source files

* (load "src/cyco")
After the source files load, switch to the CYCO package.

* (CYCO)

While using CYCO the Lisp prompt changes to :CYCO.  The exact prompt text
changes to reflect the current project name.

Optional Configuration

CYCO may be customized by loading plugins to add additional functionality.  One of the most important uses for plugins is to define the local MIDI environment.  The 'general-midi' plugin is provided as a default fallback.  Unless otherwise noted all of the example projects use the general-midi plugin.nbsp; To load the general-midi plugin use:

CYCO: (plugin 'general-midi)

Snapshots

Once CYCO is configured you may wish to create an executable snapshot.

CYCO: (save-snapshot "name-of-snapshot")

After creating the snapshot CYCO will terminate. This is normal.
